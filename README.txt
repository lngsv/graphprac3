10 - база

- три типа фигур: куб, треугольник, тетрэдр
- три различных цвета: текстура, меняющийся цвет у тетраэра, зеленый
- один объект с текстурой: треугольник металлический
- три различных объекта: куб, треугольник, тетраэдр
- хотя бы 1 источник света: источник за наблюдателем (справа сверху)
- хотя бы один объект должен двигаться: двигаются все три
